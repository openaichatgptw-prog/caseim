# App Streamlit - Precios CNH

## Ejecutar

1. Instalar dependencias:

```bash
pip install -r requirements.txt
```

2. Lanzar la app:

```bash
streamlit run app.py
```

## Funcionalidades

- Búsqueda por referencia principal/normalizada.
- Consulta de precio prorrateado y precio por origen.
- Disponibilidad consolidada.
- Última compra registrada.
- Histórico por origen.
- Últimas ventas (si existe `ventas_raw`).
- Botón para ejecutar los pipelines 01, 02 y 03.
